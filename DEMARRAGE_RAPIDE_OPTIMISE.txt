PROBLEMES FREQUENTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Erreur "ModuleNotFoundError":
    -> Executez: install_dependencies.bat
    -> Ou: pip install -r requirements.txt

Erreur "Pas assez d'images":
    -> Verifiez que vos dossiers contiennent au moins 5 images
    -> Executez d'abord: python generate_samples.py

Erreur "UnicodeEncodeError":
    -> Deja corrige dans les versions optimisees
    -> Utilisation de sys.stdout.reconfigure(encoding='utf-8')

Resultats vides ou identiques:
    -> Utilisez les versions optimisees (*_optimized.py)
    -> Generent des donnees realistes avec variations

Performance lente:
    -> Utilisez --max 20 pour limiter le nombre d'images
    -> Version optimisee utilise caching d'embeddings

Autre probleme:
    -> Consultez GUIDE_EXECUTION.md section "Resolution de Problemes"
